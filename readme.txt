== Description ==

This plugin embeds the arcRes Booking Widget on a travel supplier website and ensures it is displayed for all arcRes-driven referrals.

arcRes booking and reservations widgets can be used by hotels and tourism operators of all types (apartment bookings, condo reservations, villa rentals, activities, car rentals, tour reservations, etc.).

The widgets can be customised to your website giving travellers a consistent experience while offering advanced shopping services including interactive quotes, reservations, bookings, lead generation and direct marketing.

The widgets are easily customised to your site colours and fonts by editing a simple stylesheet file.

For more information on the arcRes system see the media kit at http://arcres.com/mediakit.pdf or e-mail support@axses.com.

== Installation ==
1. Download the plugin and unzip the arcres-booking-engine.zip file to your '/wp-content/plugins/' directory 
OR Go to the 'Plugins' menu in WordPress Admin, select 'Add New', search for 'arcResBookingWidget' and install.
2. Activate the plugin through the 'Plugins' menu in WordPress Admin
3. Set the plugin settings through the 'Settings' > 'arcRes Booking Widget' menu in WordPress Admin
4. Customize the stylesheet to match your website colors, font and layout in the appropriate stylesheet in /wp-content/plugins/arcResBookingWidget/css
5. To use the plugin - place &lt;?php arcRes_showBookingWidget(); ?&gt; in your templates where you wish the booking widget to show. You may wish to encapsulate this in &lt;div id="arcResBookingWidgetContainer"&gt;&lt;/div&gt; to further customize the look of the plugin
6. To use the widget - go to 'Appearance' > 'Widgets' in WordPress Admin and drop the widget into the desired Area(s)

= What is arcRes? =

arcRes is a marketing platform, publishing system and booking engine for hotels, villas, condos, apartments, car rentals, tourism activities and destinations. 

arcRes includes a website booking engine, a central reservation system, a travel publishing system, an interactive advertising system and technology to help manage and market travel.

arcRes publishes rates, specials and packages to an international network of sites including social sites, blogs, mobile phones, affiliated travel sites and direct to travelers.
